**Speed 공식**
```
EffectiveSpeed = min(Speed, 5) + sqrt(max(0, Speed - 5)) * 0.8
```
- Speed 5 이하: 선형 증가
- Speed 5 초과: 제곱근으로 증가폭 감소

### 2026-03-27: Speed 시스템 재설계 & 몬스터 스킬 밸런스 ✅

#### 문제점
1. Speed 6~7만 추가해도 턴을 3번 연속 잡음
2. 장비 없이 일반 몬스터 승률 100% (너무 쉬움)
3. 몬스터 스킬이 밸런스에 반영되지 않음
4. 전투 횟수에 따른 난이도 상승 없음

#### 적용 내역

**Speed 공식 도입 ✅**
```
EffectiveSpeed = min(Speed, 5) + sqrt(max(0, Speed - 5)) * 0.8
```

- Speed 5 이하: 선형 증가
- Speed 5 초과: 제곱근으로 증가폭 감소
- Speed 15에서 1.5:1 턴 비율 달성
