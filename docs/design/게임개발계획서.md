## 🗡️ 장비 시스템

### 장비 종류

| 종류 | 스킬 타입 | 설명 |
|------|-----------|------|
| **무기** | 액티브 스킬만 | 공격력, 마공 중심 |
| **갑옷** | 패시브 스킬만 | HP, 방어력 중심 |
| **악세서리** | 액티브 또는 패시브 | 밸런스 스탯 |

### 장비 등급 시스템

- 같은 장비라도 등급에 따라 스탯이 다름
- 예: 나무검(일반) → 나무검(희귀) → 나무검(에픽) → 나무검(전설)

| 등급 | 스탯 배율 | 스킬 확정 |
|------|-----------|-----------|
| **일반** | ×1.0 | 없음 |
| **희귀** | ×1.3 | 스킬 1개 |
| **에픽** | ×1.6 | 스킬 1개 (상위 스킬 확률) |
| **전설** | ×2.0 | 스킬 1개 (최상위 스킬 확률) |

### 장비 스킬 시스템

- 스킬은 장비 생성 시 **랜덤**으로 부여
- 무기: 액티브 스킬 풀에서 랜덤
- 갑옷: 패시브 스킬 풀에서 랜덤
- 악세서리: 액티브 또는 패시브 스킬 풀에서 랜덤

### 장비 스탯

- HP / 공격력 / 방어력
- MP / 마법공격 / 마법방어
- 속도 / 크리티컬

### 장비 획득

- 상자 / 전투 보상 / 상점

### 인벤토리

- 무제한

| 이름 | 타입 | 효과 |
|------|------|------|
| **파워 슬래시** | 공격 | 적에게 150% 물리 데미지 |

| **치유** | 버프/힐 | 자신의 HP 30 회복 |
| **철벽방어** | 방어 | 1턴간 받는 데미지 50% 감소 |

### 패시브 스킬 (8종) - 갑옷, 악세서리에 장착

| 이름 | 타입 | 효과 |
|------|------|------|
| **강철 의지** | 스탯 증가 | 방어력 +20% |
| **전투의 함성** | 스탯 증가 | 공격력 +15% |
| **마나 흐름** | 스탯 증가 | MP +30 |
| **재생** | 턴당 회복 | 매 턴 HP 5 회복 |
| **마나 회복** | 턴당 회복 | 매 턴 MP 3 회복 |
| **긴급 회복** | 조건부 발동 | HP 20% 이하시 전투 중 1회 HP 30 회복 |
| **반격** | 조건부 발동 | 공격받을시 30% 확률로 반격 |
| **최후의 저항** | 조건부 발동 | HP 30% 이하시 방어력 +50% |

## 🏰 던전 시스템

- **시점**: 탑뷰 (타일맵)
- **이동**: 자유 이동
- **전투 트리거**: 랜덤 인카운터 (카운트 게이지 기반, 숨김)
- **구성**: 던전 탐사 → 보스방 → 클리어
- **회복 포인트**: 던전 내 배치
- **상점**: 방문마다 랜덤 아이템

---

## ⚔️ 전투 시스템

- **시점**: 사이드뷰 (전투 씬 전환)
- **턴 순서**: 속도 스탯 기반
- **행동**: 공격 / 방어 / 스킬 (장비에서 최대 3개) / 아이템
- **패시브**: 장비 착용 시 자동 적용
- **상태이상**: 중독 / 스턴 / 화상
- **속성 상성**: 불 > 풀 > 물 > 불 (데미지 ×1.5 / ×0.5)

## 🎯 게임 플로우

```
타이틀
   ↓
던전 입장 (상점 이용 가능)
   ↓
던전 탐사 (자유 이동)
   ↓
카운트 게이지 → 전투 (사이드뷰)
   ↓
승리 → 장비/골드 획득
패배 → 게임 오버 (처음부터)
   ↓
보스방 도달
   ↓
보스 처치 → 클리어 보상 (장비 + 골드)
```
